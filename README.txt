The objective of the game is to reach the corner three spot indicated by the green marker. The game provides real-time feedback regarding the distance of the basketball from the target position. The basketball is controlled by the arrow keys that the user uses. The controls of the program include:
1. Enter - to start game
2. Backspace - to restart game
3. Up Arrow - move ball up
4. Down Arrow - move ball down
5. Right Arrow - move ball right
6. Left Arrow - move ball left
